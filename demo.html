jQuery Showit

A plugin for jQuery to show any element centered

To display data with this plugin, all you have to call is .showit(); on any element and you're set. There are however several arguments you can pass along to the plugin to help you make the experience even more awesome!

Arguments:

  • borderSize: This overrides the CSS for the element you're calling and sets a border acording to borderSize and borderColor.
    default: 0px
  • borderColor: See borderSize above.
    default: #000
  • closeButton: A boolean to display the close button.
    default: false
  • closeButtonText: The text on the close button if it's shown.
    default: Close
  • fadeSpeed: The speed of which to fade the element in and out, in ms.
    default: 400
  • fontSize: The font-size to display the title and close button with.
    default: .8em
  • overlayAutoClick: If the overlay plugin is included in the project, this will forward the autoClick setting to it.
    default: true
  • overlayOpacity: Same as above with the overlay plugin, this time the opacity setting.
    default: 0.5
  • title: The title to display below the element along with the close button.
    default: The title-attribute of the called element
  • titleColor: The color to apply to the title string being shown.
    default: #000
  • titleBgColor: The background color to apply to the frame containing the title and close button.
    default: #ccc
  • zIndex: The elements place in the order of things. Best not tamper with this one.
    default: 999

Examples:

  • .showit();
  • .showit({ borderSize: 10, borderColor: '#0f0' });
  • .showit({ closeButton: true, closeButtonText: 'Shut me down dude' });
  • .showit({ fadeSpeed: 5000 });
  • .showit({ fontSize: '3em', title: 'DUDE!!!', titleColor: '#f00', titleBgColor: '#ff0' });

jQuery.showit

jQuery plugin to show any element centered on the visible screen Author: Stian Hanger (pdnagilum@gmail.com)

Settings

  • fadeSpeed - int - optional - default: 400 The speed at which the element is to fade in. Setting 0 will make it instant.
  • zIndex - int - optional - default: 999 The CSS z-index to be set on the element.
  • css - json - optional Other CSS settings to be passed to the element.
  • overlayUseIfLoaded - boolean - optional - default: false Use overlay if it is loaded.
  • overlayObject - object - optional - default: document The javascript object to attach overlay too.
  • overlayOptions - json - optional An array of settings to pass along to overlay.

Try it

  • .showit() The most basic call. This will display the called element and show it centered.
  • .showit({ fadeSpeed: 4000 }) Here the fade speed is greatly increased, slowing the fade-in.
  • .showit({ css: { 'background': '#eee', 'border': 'solid 5px #aaa', 'padding': '10px' } }) Here we pass along CSS style to the element.

Try it with overlay

overlay is a jQuery plugin to add an overlaying effect, which works great with showit. Read more about it on https://github.com/egilkh/jquery.overlay
    .showit({ overlayUseIfLoaded: true, css: { 'background': '#eee', 'border': 'solid 5px #aaa', 'padding': '10px' } }) Basic call with overlay.
